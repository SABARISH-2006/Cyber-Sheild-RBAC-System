SECTION 5: HOW EMAIL SYSTEM WORKS
═══════════════════════════════════════════════════════════════════════════════

FLOW 1: Password Recovery Email
   User clicks → "Forgot Password"
   ↓
   Enters email → premkumarsanjay2006@gmail.com
   ↓
   System generates → Reset token (1 hour expiration)
   ↓
   Calls → send_reset_email()
   ↓
   Brevo API receives request
   ↓
   Email sent → HTML formatted with reset button
   ↓
   User clicks link → Resets password

FLOW 2: MFA Verification Email
   User logs in → With valid credentials
   ↓
   System generates → 6-digit OTP (5 min expiration)
   ↓
   Calls → send_otp_email()
   ↓
   Brevo API receives request
   ↓
   Email sent → Large easy-to-read OTP code
   ↓
   User enters code → Completes login
