HOW IT WORKS NOW:

Recovery Email Flow:
   User clicks "Forgot Password"
   → System generates reset token (1 hr expiry)
   → Calls send_reset_email()
   → Brevo API sends HTML email
   → User receives beautiful email with reset button
   → Clicks button → Resets password

MFA OTP Email Flow:
   User logs in with valid credentials
   → System generates 6-digit OTP (5 min expiry)
   → Calls send_otp_email()
   → Brevo API sends HTML email
   → User receives email with large OTP code
   → Enters code → Completes login
